## Description

Movie Ticket Booking API built using NestJs

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```

## Avaliable Endpoints

- /movies : get list of movies

- /movies/{id} : get movie by id

- /movies/book-movie-ticket : book movie ticket
